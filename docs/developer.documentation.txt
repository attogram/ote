Open Translation Engine (OTE)

Developer Documentation
-----------------------

* Thank you!  

  The OTE depends on the contribution of developers like yourself.


* Our mission:

  The Open Translation Engine is an open source project 
  building a web-based system to enable community management 
  of translation dictionaries.


* Submitting bug reports:

  Please create an account on sourceforge.net if you do not yet have one.
  
  email to sourceforge user 'meta_nl'
  http://sourceforge.net/users/meta_nl/


* Joining the discussion:

  Project Home: http://sourceforge.net/projects/ote/


* Getting the latest version from SVN:

  svn co https://ote.svn.sourceforge.net/svnroot/ote ote 


* Getting on the roadmap:

  /docs/roadmap.txt
  /docs/changelog.txt


* Submitting patches:
  
  email to sourceforge user 'meta_nl'
  http://sourceforge.net/users/meta_nl/


* Getting SVN commit access:

  SVN commit access is restricted to active developers 
  who have proven they work well with the overall OTE community.
