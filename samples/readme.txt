
Welcome to the Dictionary Additions Management System (DAMS),
a collection of open source translation dictionary files

More Info:
http://sourceforge.net/projects/dams/

SVN access:
https://dams.svn.sourceforge.net/svnroot/dams

Download latest version as Gnu Tarball:
http://dams.svn.sourceforge.net/viewvc/dams.tar.gz?view=tar


These files are compatible with the Open Translation Engine (OTE).
For more info, see http://sourceforge.net/projects/ote/
